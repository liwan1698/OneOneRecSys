# OneOneRecSys
这是一个推荐系统项目。项目的目标是包含常见推荐场景，包括短视频推荐、电影推荐和新闻推荐。当前正在开发短视频推荐。<br>系统使用flink和sparkML作为实时推荐引擎，sparkML
作为离线推荐引擎。根据用户的个人信息和产品信息使用协同过滤进行召回，之后使用deepFM进行排序,返回推荐结果。<br>

# Q&A
* *如果获取用户个人信息？*<br>
app使用vue编写，不能获取用户的地理位置、年龄、性别等个人信息，所以随机生成用户的这些信息。
* *如何获取短视频？*<br>
通过爬虫爬取部分短视频作为系统的视频库。
